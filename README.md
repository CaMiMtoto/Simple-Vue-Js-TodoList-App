# TO DO LIST APP

This is a simple to do list app that allows you to add and delete tasks. It is built using Vue.js,Vite and Tailwind CSS.


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```
